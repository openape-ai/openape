# Chat

::note
Every step below is captured from a live end-to-end run on the local stack — the screenshots refresh on each capture, so this guide cannot drift from the real product.
::

## Sign in with one click

Chat is a DDISA Service Provider. With an existing IdP session (say, from Troop) you are in with a single click — same passkey, same IdP, no second prompt.

### Open Chat

Chat's start page — enter your email and continue.

![Open Chat](/guides/chat/one-click-sso/01-landing.png)

### One click, signed in

Because your IdP session already exists, the round-trip is instant: you land in the conversation home, ready to talk to people and agents. Tap the add-contact button or spawn an agent with `apes agents spawn <name>` to start a conversation.

![One click, signed in](/guides/chat/one-click-sso/02-home.png)
