# PR

::note
Every step below is captured from a live end-to-end run on the local stack — the screenshots refresh on each capture, so this guide cannot drift from the real product.
::

## Review an agent's pull request

When an agent opens a PR it lands here for a human to approve before anything merges. Sign in once and every agent diff waits for your verdict.

### Sign in to PR

Your review inbox: every PR an agent opened, newest first, each tagged with who authored it (🤖 for an agent) and its diff size.

![Sign in to PR](/guides/pr/review-an-agent-pr/01-inbox.png)

### Read the diff and decide

The full diff rendered file by file, additions and deletions side by side. Approve or request changes — the agent is polling for your verdict and continues the moment you decide.

![Read the diff and decide](/guides/pr/review-an-agent-pr/02-review.png)
